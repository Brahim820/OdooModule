Customer Payment Receipt

Module provides utility to print payment receipt of customers and suppliers.



The Graphical overview

Payment receipt will present all needful information on report.

Print multiple receipts together

From list view of the Customer/Supplier Payments, user can select multiple Payments for which he/she wants to generate the report. Then go to Print > Print Payment Receipt.

Print report for single customer

From Form view of the Customer/Supplier Payments, user can print report by going to Print > Print Payment Receipt.
